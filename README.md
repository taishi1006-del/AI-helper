# AI文章メーカー

AIを初めて使う人でも、用途を選んで空欄を埋めるだけでAI向けの文章を作れる補助ツールです。

## 開き方

`index.html` をブラウザで開いてください。インターネット接続や追加インストールは不要です。

## できること

- メール文を作る
- 文章を要約する
- 難しい内容を説明してもらう
- アイデアを出す
- 作業手順を作る
- 文章を直す
- 質問文を作る
- 文章を添削する
- 文全体を敬語にする
- 文章の印象を変える
- 比較して質問する
- 手順を聞く
- 原因を相談する

作成された文章は、コピーしてChatGPTなどのAIサービスに貼り付けて使えます。
