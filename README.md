# レポートAIサポーター

学生がレポートを書くときに、AIへ相談するための質問文を作れる補助ツールです。
AIに丸ごと書かせるのではなく、テーマ整理、構成、資料確認、下書き、添削、引用確認を支援する方針で作っています。

## 開き方

`index.html` をブラウザで開いてください。インターネット接続や追加インストールは不要です。

## できること

- レポートテーマを絞る
- 問いと仮説を作る
- 序論・本論・結論の構成を作る
- 調査キーワードを作る
- 資料メモを整理する
- 序論、本論、結論の書き方を相談する
- レポート文を添削する
- 論理のつながりを確認する
- 引用と参考文献を確認する
- AI利用範囲を整理する
- 画像を添付して観察点や判別結果を整理する
- グラフや表の画像を読み取る
- 資料画像や板書画像をメモ化する
- 自分だけのレポート用テンプレートを作る

作成された文章は、コピーしてChatGPTなどのAIサービスに貼り付けて使えます。
画像判別テンプレートでは、画像プレビューで確認したあと、作成された質問文と同じ画像をAIサービスに添付して使います。

自作テンプレートは、使っているブラウザ内に保存されます。同じ端末とブラウザで開くと、次回も続けて使えます。
